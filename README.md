# Discord Bot Final

Ready to deploy on Railway or any Node.js environment.

## Setup

1. Add environment variables:
   - DISCORD_TOKEN
   - GEMINI_API_KEY
   - GITHUB_TOKEN
   - GITHUB_REPO
   - GITHUB_FILE
   - BOT_OWNER_ID

2. Install dependencies:
   ```
   npm install
   ```

3. Start the bot:
   ```
   npm start
   ```
